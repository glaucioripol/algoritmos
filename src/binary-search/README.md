## Exercícios

1 - Em uma lista com 128 itens, qual será o maior número de etapas para encontrar um item usando busca binária?

Passo a passo:
128/2 = 64
64/2 = 32
32/2 = 16
16/2 = 8
8/2 = 4
4/2 = 2
2/2 = 1

Resposta:
Total de 7 interações
128 = 2^7
